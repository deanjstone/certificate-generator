# Feature Specifications

## Core Features

### Certificate Generation
- Upload an Excel spreadsheet with learner data
- Parse learner name and completed units from the spreadsheet
- Generate a formatted PDF certificate
- Download or save the generated certificate

### Data Format (Excel Input)
- `A1`: Learner name (required)
- `A2:B11`: Unit code (col A) and unit title (col B), up to 10 units
